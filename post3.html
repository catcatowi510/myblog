HTML & CSS - Nền Tảng Web

Hướng dẫn cơ bản về HTML và CSS, những công nghệ nền tảng để xây dựng giao diện web đẹp mắt

HTML - Ngôn Ngữ Đánh Dấu Siêu Văn Bản

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu được sử dụng để tạo cấu trúc cho các trang web. HTML định nghĩa nội dung của trang web thông qua các thẻ (tags).

Cấu Trúc HTML Cơ Bản

<!DOCTYPE html>
<html lang="vi">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Trang Web Của Tôi</title>
</head>
<body>
    <header>
        <h1>Chào Mừng Đến Với Website</h1>
        <nav>
            <ul>
                <li><a href="#home">Trang Chủ</a></li>
                <li><a href="#about">Giới Thiệu</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <section>
            <h2>Nội Dung Chính</h2>
            <p>Đây là đoạn văn bản mô tả.</p>
        </section>
    </main>
    
    <footer>
        <p>© 2024 Bản Quyền</p>
    </footer>
</body>
</html>

Các Thẻ HTML Quan Trọng

  • <h1> - <h6>: Tiêu đề từ lớn đến nhỏ
  • <p>: Đoạn văn bản
  • <a>: Liên kết
  • <img>: Hình ảnh
  • <div>: Container block
  • <span>: Container inline
  • <ul>, <ol>, <li>: Danh sách
  • <form>, <input>: Form nhập liệu

CSS - Cascading Style Sheets

CSS là ngôn ngữ được sử dụng để định dạng và tạo kiểu cho các trang web. CSS giúp trang web trở nên đẹp mắt và hấp dẫn hơn.

3 Cách Sử Dụng CSS

1. Inline CSS

<p style="color: blue; font-size: 18px;">Văn bản màu xanh</p>

2. Internal CSS

<head>
    <style>
        p {
            color: blue;
            font-size: 18px;
        }
    </style>
</head>

3. External CSS (Khuyên dùng)

<head>
    <link rel="stylesheet" href="style.css">
</head>

CSS Selectors

/* Element selector */
p {
    color: black;
}

/* Class selector */
.my-class {
    background-color: yellow;
}

/* ID selector */
#my-id {
    font-size: 20px;
}

/* Descendant selector */
div p {
    margin: 10px;
}

/* Pseudo-class */
a:hover {
    color: red;
}

CSS Box Model

Box Model là khái niệm quan trọng trong CSS, mô tả cách các phần tử HTML được hiển thị:

  • Content: Nội dung thực tế
  • Padding: Khoảng cách giữa content và border
  • Border: Đường viền
  • Margin: Khoảng cách bên ngoài
.box {
    width: 300px;
    height: 200px;
    padding: 20px;
    border: 2px solid black;
    margin: 15px;
    background-color: lightblue;
}

Flexbox - Layout Hiện Đại

Flexbox giúp tạo layout linh hoạt và responsive dễ dàng:

.container {
    display: flex;
    justify-content: center;  /* Căn giữa theo chiều ngang */
    align-items: center;      /* Căn giữa theo chiều dọc */
    gap: 20px;                /* Khoảng cách giữa các item */
}

.item {
    flex: 1;  /* Các item chiếm đều không gian */
}

CSS Grid - Layout Mạnh Mẽ

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 20px;
}

/* Responsive */
@media (max-width: 768px) {
    .grid-container {
        grid-template-columns: 1fr;
    }
}

Responsive Design

Tạo website responsive với Media Queries:

/* Mobile First Approach */
.container {
    width: 100%;
    padding: 10px;
}

/* Tablet */
@media (min-width: 768px) {
    .container {
        width: 750px;
        margin: 0 auto;
    }
}

/* Desktop */
@media (min-width: 1200px) {
    .container {
        width: 1140px;
    }
}

CSS Animations

/* Transition */
.button {
    transition: all 0.3s ease;
}

.button:hover {
    transform: scale(1.1);
    background-color: blue;
}

/* Keyframe Animation */
@keyframes fadeIn {
    from {
        opacity: 0;
        transform: translateY(20px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

.element {
    animation: fadeIn 1s ease-in-out;
}

Tips Học HTML & CSS

  1. Thực hành thường xuyên: Code mỗi ngày để cải thiện kỹ năng
  2. Clone các website: Học bằng cách copy design các trang nổi tiếng
  3. Sử dụng DevTools: Inspect element để học cách người khác code
  4. Học Flexbox và Grid: Hai công cụ layout quan trọng nhất
  5. Mobile First: Luôn thiết kế cho mobile trước
  6. Validation: Kiểm tra code với W3C Validator

Kết Luận

HTML và CSS là nền tảng cơ bản nhất của web development. Dù có học framework nào đi nữa (React, Vue, Angular), bạn vẫn cần nắm vững HTML & CSS. Hãy thực hành nhiều và tạo các project thực tế để nâng cao kỹ năng!
