React - Thư Viện UI Hàng Đầu

Chia sẻ kinh nghiệm học và sử dụng React để xây dựng ứng dụng web hiện đại

React Là Gì?

React là một thư viện JavaScript được phát triển bởi Facebook (Meta) để xây dựng giao diện người dùng. React cho phép tạo các ứng dụng web tương tác cao với hiệu suất tốt thông qua Virtual DOM và component-based architecture.

Tại Sao Nên Học React?

  • Phổ biến nhất: Được sử dụng bởi Facebook, Instagram, Netflix, Airbnb...
  • Component-based: Tái sử dụng code dễ dàng
  • Virtual DOM: Render nhanh và hiệu quả
  • Ecosystem lớn: Nhiều thư viện và công cụ hỗ trợ
  • React Native: Có thể làm mobile app
  • Community mạnh: Dễ tìm tài liệu và support

React Hooks

useState - Quản Lý State

import { useState } from 'react';

function Counter() {
    const [count, setCount] = useState(0);
    
    return (
        <div>
            <p>Bạn đã click {count} lần</p>
            <button onClick={() => setCount(count + 1)}>
                Click me
            </button>
        </div>
    );
}

useEffect - Side Effects

import { useState, useEffect } from 'react';

function UserProfile({ userId }) {
    const [user, setUser] = useState(null);
    const [loading, setLoading] = useState(true);
    
    useEffect(() => {
        // Fetch data khi component mount
        fetch(`/api/users/${userId}`)
            .then(res => res.json())
            .then(data => {
                setUser(data);
                setLoading(false);
            });
        
        // Cleanup function
        return () => {
            console.log('Component unmount');
        };
    }, [userId]); // Dependencies array
    
    if (loading) return <div>Loading...</div>;
    
    return <div>{user.name}</div>;
}

useContext - Share Data

import { createContext, useContext, useState } from 'react';

// Tạo Context
const ThemeContext = createContext();

// Provider
function ThemeProvider({ children }) {
    const [theme, setTheme] = useState('light');
    
    return (
        <ThemeContext.Provider value={{ theme, setTheme }}>
            {children}
        </ThemeContext.Provider>
    );
}

// Sử dụng Context
function ThemedButton() {
    const { theme, setTheme } = useContext(ThemeContext);
    
    return (
        <button onClick={() => setTheme(theme === 'light' ? 'dark' : 'light')}>
            Current theme: {theme}
        </button>
    );
}

Props và State

// Parent Component
function App() {
    const [message, setMessage] = useState('Hello');
    
    return (
        <ChildComponent 
            message={message}
            onUpdate={(newMsg) => setMessage(newMsg)}
        />
    );
}

// Child Component
function ChildComponent({ message, onUpdate }) {
    return (
        <div>
            <p>{message}</p>
            <button onClick={() => onUpdate('Updated!')}>
                Update
            </button>
        </div>
    );
}

Form Handling

function LoginForm() {
    const [formData, setFormData] = useState({
        email: '',
        password: ''
    });
    
    const handleChange = (e) => {
        setFormData({
            ...formData,
            [e.target.name]: e.target.value
        });
    };
    
    const handleSubmit = (e) => {
        e.preventDefault();
        console.log('Form submitted:', formData);
    };
    
    return (
        <form onSubmit={handleSubmit}>
            <input
                type="email"
                name="email"
                value={formData.email}
                onChange={handleChange}
                placeholder="Email"
            />
            <input
                type="password"
                name="password"
                value={formData.password}
                onChange={handleChange}
                placeholder="Password"
            />
            <button type="submit">Login</button>
        </form>
    );
}

Best Practices

  1. Component nhỏ gọn: Mỗi component chỉ làm một việc
  2. Sử dụng hooks đúng cách: Không call hooks trong loops/conditions
  3. Key cho lists: Luôn dùng unique key khi render list
  4. Tránh inline functions: Có thể gây re-render không cần thiết
  5. PropTypes hoặc TypeScript: Type checking cho components
  6. Code splitting: Lazy load components để tối ưu performance

Thư Viện Hữu Ích

  • React Router: Navigation và routing
  • Redux/Zustand: State management
  • React Query: Data fetching
  • Tailwind CSS: Styling nhanh
  • React Hook Form: Form handling
  • Framer Motion: Animations

Kết Luận

React là một công cụ mạnh mẽ để xây dựng UI hiện đại. Với component-based architecture và hooks, React giúp code dễ maintain và scale. Hãy thực hành nhiều bằng cách làm các project thực tế để nắm vững React!
